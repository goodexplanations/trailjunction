do you know the way?

💡 The Idea

Website which fundamentally has: a map view with each location of a trail run plotted on it. (Later version, when you zoom in, see actual trail that you run on) Then some sliders: date, distance and a quick select for a province, slider for entry price, multi-day or not, etc. A different version of this could be for places to run, not necessarily races:

  • At first glance it should show Distance, Elevation, Terrain and Saffer Trails Score
  • Sliders would be access: public, paid, private
  • Links out to Event Pages etc.

Features to Build

Extra features that could be added:

  • Ability to connect your strata and it gives you an estimated time
  • Monetise by linking to websites
  • You can enter your location and it gives “distance from you” - similar to AutoTrader for car sales
  • “Need someone to go with?” - refer to a trail guide
  • User login, that’s a whole new universe
